# Lucene Python Wrapper
* Author: Bonggun Shin

## Installation
* Compilation 

```bash
git clone git@github.com:bgshin/luceneInterface.git
cd luceneInterface
ant clean build jar
ls ./bin/luceneInterface.jar 
```

## Set classpath
* add the location of "luceneWrapper.jar" to ~/.bashrc or ~/.bash_profile
